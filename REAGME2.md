# Predictive Modeling Using Machine Learning

## Project Overview
This project predicts student marks based on study hours and attendance using Linear Regression.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-Learn

## Features
- Data preprocessing
- Train-test split
- Model training
- Prediction
- Performance evaluation

## Dataset
Hours Studied, Attendance, and Marks.

## Conclusion
A simple machine learning project demonstrating predictive modeling using Linear Regression.
