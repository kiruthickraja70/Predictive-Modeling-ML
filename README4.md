# Retail Sales Data Analysis

## Project Overview
This project analyzes retail sales data to identify trends, product performance, and business insights.

## Objective
- Analyze sales data.
- Calculate total sales.
- Identify the highest-selling category.
- Visualize sales using charts.

## Technologies Used
- Python
- Pandas
- Matplotlib

## Features
- Data Analysis
- Statistical Summary
- Category-wise Sales
- Product-wise Sales
- Bar Chart
- Pie Chart

## Results
- Electronics generated the highest revenue.
- Laptop is the top-selling product.
- Visualizations help understand sales performance.

## Conclusion
This project demonstrates how data analysis can help retail businesses make better decisions.
